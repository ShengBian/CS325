# Project Description

This is homework 5 for OSU CS325.

Description: This program read input from a file contains the number
of wrestlers, n, followed by their names, the number of rivalries r
and rivalries listed in pairs. It will output whether it is possible
to designate some of the wrestlers as Babyfaces and the remainder as
Heels such that each rivalry is between a Babyface and a Heel.

# Getting Started
On flip, you need to first put data file such as "wrestler.txt" to the same foler of babyfaces_vs_heels.py.
Then, you need to enter "python babyfaces_vs_heels.py".
The program will ask you "Enter a filename: ", you need to enter the name of data file, such as "wrestler.txt".
Then, it will display the results. 

# Result
Output: Either of results are outputted to the terminal.
Yes, if possible followed by a list of the Babyface wrestlers and a list of the Heels .
No, if impossible.
